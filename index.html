QUERY_LOGS

Who is eligible for HackerGenesis? +
Any university student or independent developer with a passion for CSE and problem-solving. Teams of 1-4 are allowed.
What is the registration cost? +
Access is completely free. We provide the infrastructure, APIs, and food (for in-person) - you provide the innovation.
Can I submit an existing project? +
No. All code must be written during the 48-hour event window. Open source libraries are allowed, but the core logic must be fresh.
Are there hardware requirements? +
A laptop and a charger are all you need. We provide cloud credits for high-compute tasks like model training.
